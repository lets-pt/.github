## 핵심기능
<ul>
  <li>
    연습모드 - 음성 / 표정인식 AI를 통한 실시간 피드백
  </li>
  <li>
    실전모드 - 실시간 양방향 소통을 통한 다른 유저의 피드백
  </li>
  <li>
    연습기록 - 연습기록을 남겨두어 실시간이 아니더라도 다른 유저들이 언제든지 피드백 가능
  </li>
</ul>
